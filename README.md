## Tech Stack
- Android Studio (Kotlin)
- Clean Architecture
- Dynamic Feature (Search)
- Room 
- Retrofit
- Okhttp3
- Glide
- CI/CD (Code Coverage, Code Style, dan Vulnerability)
- Dependency Injection (Koin)
- Leak Canary
- Obfuscation with ProGuard
- Encryption Database
- Certificate Pinning
- NetworkBoundResource
- RxJava
API : TheMovieDB
